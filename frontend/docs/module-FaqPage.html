FaqPage

Ce composant affiche les questions fréquemment posées (FAQ) sur l'application DreamHabitat, organisées par catégories avec des sections dépliables pour une meilleure lisibilité. Il couvre les sujets généraux, la tarification, les aspects techniques, la gestion du compte et le support.
